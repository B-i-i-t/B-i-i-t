# B-i-i-t

インフラ・Linux・自宅サーバー運用を中心に、運用自動化ツールやWebアプリケーションを制作しています。  
自宅サーバー、Tailscale、Docker、Shell scripting、Webアプリケーション開発に関心があります。

## Focus

- Linux / HomeLab
- Docker / Docker Compose
- Tailscale
- Shell scripting
- Ansible
- Java / Servlet / JSP
- JavaScript / TypeScript
- Azure Functions

## Featured Projects

### AgentPad
iPadからLinux開発環境を操作するためのWebベースツールです。  
PTY接続、ファイル閲覧、git diff表示、PWA、Docker Compose構成に対応しています。

- Linux開発環境の遠隔操作
- WebSocket / PTY
- Docker Compose
- PWA

### Tailscale Waybar Module
Tailscaleの接続状態をWaybarに表示するShell製モジュールです。  
Linuxデスクトップ環境でVPN接続状態を確認しやすくするために作成しました。

- Shell scripting
- Tailscale
- Waybar
- Linux desktop

### School Automation Tool
GitHub Pages、Azure Functions、Gemini APIを利用した学校作業支援ツールです。  
フロントエンドとサーバーレスAPIを分離し、APIキーはバックエンド側で管理しています。

- GitHub Pages
- Azure Functions
- Gemini API
- CORS
- GitHub Secrets

### Score Management System
Java Servlet/JSP、H2 Database、BCryptを用いた成績管理Webアプリケーションです。  
ログイン、学生管理、科目管理、成績管理などの基本的なCRUD機能を実装しています。

- Java 21
- Jakarta Servlet / JSP
- H2 Database
- BCrypt
- Maven / Tomcat
